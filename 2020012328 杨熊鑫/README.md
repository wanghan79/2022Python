  - MCC的计算公式为$MCC=\frac{TP\times TN-FP\times FN}{\sqrt{(TP+FP)(TP+FN)(TN+FP)(TN+FN)}}$，当数据规模较大时，由于python生成的是伪随机数，不会出现上述计算公式分母为0的情况。但是由于设置了可以输入的数据规模，可能在较小输入规模时，出现分母为0的情况，这个时候$MCC=0$，需要特殊判定。
    ```python
    denominator = math.sqrt((TP + FP) * (TP + FN) * (TN + FP) * (TN + FN))
    # when the denominator equals 0, mcc equals 0.
    if denominator == 0:
      mcc = 0
    else:
      mcc = (TP * TN - FP * FN) / denominator
    ```
